# guess-the-number

## Overview
This is a simple interactive game built with **HTML, CSS, and JavaScript**. The player tries to guess a random number generated by the computer. The game includes difficulty levels that can be adjusted dynamically.

## Features
- Random number generation with Math.random()
- Dynamic difficulty levels (Easy, Normal, Hard)
- Feedback messages for win or lose
- User-friendly interface with styled buttons and input
- Color-coded difficulty display

## How to Play
1. Enter a number in the input field.
2. Press the "Press" button to check your guess.
3. Adjust difficulty with the "+" and "-" buttons (changes range from 1 to the selected limit).
4. Try to guess the number generated by the computer.

## Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
